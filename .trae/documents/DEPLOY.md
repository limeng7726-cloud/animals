# 部署指南

恭喜你完成了动物主题个人网站的开发！现在，让我们把它发布到互联网上，让所有人都能看到。

## 推荐方案：使用 Vercel 部署（最简单、免费）

Vercel 是一个非常流行的前端托管平台，完美支持 React 项目。

### 步骤 1：准备代码
1.  确保你的代码已经全部保存。
2.  如果你还没有使用 Git，请初始化仓库并提交代码：
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```
3.  在 GitHub 上创建一个新仓库，并将本地代码推送到 GitHub。

### 步骤 2：在 Vercel 上部署
1.  访问 [Vercel 官网](https://vercel.com) 并注册账号（推荐直接用 GitHub 账号登录）。
2.  在控制台点击 **"Add New..."** -> **"Project"**。
3.  在 "Import Git Repository" 列表中找到你的 GitHub 仓库，点击 **"Import"**。
4.  在配置页面：
    *   **Project Name**: 可以自定义项目名称。
    *   **Framework Preset**: Vercel 会自动识别为 `Vite`，无需修改。
    *   **Root Directory**: 保持默认 `./`。
5.  点击 **"Deploy"** 按钮。

### 步骤 3：等待部署完成
*   Vercel 会自动构建你的项目，通常只需要几十秒。
*   构建完成后，你将获得一个类似 `https://your-project.vercel.app` 的网址。
*   点击该网址，你的网站就上线了！🎉

## 注意事项

### 关于数据保存
为了简化部署流程并确保网站稳定性，我们将数据（动物信息、照片、故事）直接集成到了网页代码中（纯前端模式）。
这意味着：
*   **浏览体验完美**：网站加载速度快，且不会出现数据加载失败的情况。
*   **互动限制**：在网页上进行的任何修改（如发表留言、添加新动物）仅在当前浏览器会话中有效，刷新页面后会重置。
*   **如何修改内容**：如果你想永久修改内容（例如添加新的动物朋友），请直接修改代码中的 `src/data/*.json` 文件，然后重新推送到 GitHub，Vercel 会自动更新你的网站。
